World of Warcraft Professions

July 2023 - Ongoing

Website to consolidate World of Warcraft character profession data.

Description

Maintaining many professions across many characters within World of Warcraft can make it difficult to keep track of the finer details. By constructing trillithia.pythonanywhere.com, I was able to create a resource that consolidated all of this information into one location that is easily accessible and traverisble. Since the game is also online, the website opens friends to this information so that they can place orders without needing me to be around.

This website was built using the Flask library in Python to run the backend, as well as HTML files and a mix of custom and Bootstrap CSS. The data is read from a collection of CSV and JSON files, processed using Python to determine which crafters can create which recipes. In the event of multiple crafters knowing a recipe, it works with the data from the crafter who can create the recipe at the highest quality.
